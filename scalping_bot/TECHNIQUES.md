# เทคนิคเทรดสั้นที่ Scalping Bot ใช้ (และที่แนะนำให้คุณเข้าใจ)

> หลักคิด: **เทรดสั้น = ตามเทรนด์ใหญ่ + เข้าจังหวะย่อ/เบรก + คุมความเสี่ยงเป๊ะ**
> ไม่ใช่เดาว่าจะขึ้น/ลง แต่ "รอจังหวะที่ความน่าจะเป็นเข้าข้าง แล้วเสี่ยงน้อยตอนผิด"

## 1) โครงเทรนด์ก่อน (Top-down) — `patterns.structure`
เทรดสั้นต้อง **ตามทิศใหญ่** เสมอ:
- ขาขึ้น = ทำ **Higher High + Higher Low** (HH/HL) → หาจังหวะ "ซื้อตอนย่อ"
- ขาลง = **Lower High + Lower Low** (LH/LL) → หาจังหวะ "ขาย/Put ตอนเด้ง"
- ออกข้าง = **ไม่เทรด** (เทรนด์ไม่ชัด = ความน่าจะเป็นไม่เข้าข้าง)
- ใช้ Multi-timeframe: TF ใหญ่บอกทิศ (D1/H4) · TF เล็กจับจังหวะ (H1/M15)

## 2) จังหวะเข้า — 2 แบบที่เวิร์กสุด
**A. ย่อแตะแนวรับแล้วเด้ง (Pullback)** — `patterns.pullback_to_ema`
ในขาขึ้น รอราคาย่อมาแตะ EMA (20/50) แล้วเด้ง = เข้าต่อเทรนด์ ความเสี่ยงต่ำ ไม่ไล่ของแพง
→ ตรงกับ "💰 โซนเข้าซื้อ" ของ Part 1 เป๊ะ

**B. เบรกกรอบสะสม (Breakout)** — `patterns.breakout`
ราคาออกข้างสะสมพลัง แล้วปิดทะลุกรอบ **พร้อมวอลุ่มเข้า** = เริ่มเทรนด์ใหม่
→ ต้องมีวอลุ่มยืนยัน ไม่งั้นเป็น **เบรกหลอก (fakeout)**

## 3) วอลุ่มยืนยัน — `patterns.volume_entering`
**"Volume เริ่มเข้า"** ที่คุณพูดถึง = วอลุ่มแท่งที่วิ่ง **มากกว่าค่าเฉลี่ย + กำลังไต่ขึ้น**
- เบรก + วอลุ่มพุ่ง = ของจริง ✅
- เบรก + วอลุ่มแผ่ว = ระวังหลอก ⚠️
- ขาขึ้นที่วอลุ่มหดตอนย่อ แล้วพุ่งตอนไปต่อ = สุขภาพดี

## 4) แท่งเทียนคอนเฟิร์ม (Trigger) — `candles.detect`
รอ "แท่งสัญญาณ" ตรงโซนเข้า อย่าเข้ามั่ว:
- **Bullish/Bearish Engulfing** — แท่งกลืน = แรงสวนชัด (สัญญาณแรง)
- **Pin bar / Hammer / Shooting star** — หางยาวปฏิเสธราคา = กลับตัว
- **Inside bar** — สะสมก่อนระเบิด → เทรดตอนเบรกออกจาก inside bar
- **Doji** ปลายเทรนด์ = ลังเล ระวังกลับตัว
- **Marubozu** = แท่งตัน แรงล้วน

## 5) Risk Management — `risk.py` (สำคัญสุด 80%)
- **ขนาดไม้** = คำนวณให้ขาดทุนที่ SL = **1-2% ของพอร์ต** (ไม่ใช่ความรู้สึก)
- **SL จากโครงสร้าง** — ใต้ swing low / ใต้ EMA / 1.5×ATR (ไม่ใช่ตั้งมั่ว)
- **R:R ≥ 1.5** — เป้าต้องไกลกว่าความเสี่ยง ไม่งั้นไม่คุ้ม (ข้าม)
- **กินกำไรบางส่วนที่ 1R → เลื่อน SL มาเท่าทุน** = ไม้นี้ไม่ขาดทุนอีกแล้ว
- **เพดานต่อวัน** — ขาดทุนรวมถึง X% = หยุด (circuit breaker กันไม้แก้แค้น)
- **ไม่ถือข้ามข่าวใหญ่** (NFP/FOMC/CPI/งบ) — สั้น = เสี่ยง gap

## 6) Gemini ด่านสุดท้าย — `gemini_gate.py`
ก่อนออกใบสั่ง ให้ AI หา "เหตุผลที่ไม้นี้จะแพ้" → จับของที่คนใจร้อนมองข้าม
(ยืดเกิน/สวนเทรนด์/ชนแนวต้าน/วอลุ่มหลอก) = ลดขาดทุนที่เลี่ยงได้

---
**กฎทอง:** เข้าเมื่อ **เทรนด์ + จังหวะ + วอลุ่ม + แท่งเทียน เห็นพ้อง** เท่านั้น
ขาดข้อใดข้อหนึ่ง = ความน่าจะเป็นลด = ไม้เล็กลงหรือข้าม
